# Jardín de Nono 🌿

App de seguimiento del jardín de Nono: mapa satelital con un pin por planta, fotos, bitácora y semáforo de estado (🟢🟡🔴). Multiusuario (Freddy, Benito) con datos en Supabase.

- **App:** `index.html` (servida por GitHub Pages)
- **Backend:** Supabase — login, base de datos y almacenamiento de fotos

Uso interno familiar.
