# Jardín de Nono — app React (fuente)

App construida con Vite + React + Tailwind. El sitio publicado (raíz del repo)
es el resultado de `npm run build` de esta carpeta.

- `npm install` — instalar dependencias
- `npm run dev` — desarrollo local
- `npm run build` — genera `dist/`; el contenido de `dist/` se copia a la raíz del repo

El app clásico (un solo archivo HTML) quedó archivado en `/clasico/`.
